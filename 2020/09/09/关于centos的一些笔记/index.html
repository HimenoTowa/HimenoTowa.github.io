安装hpctoolkit

先git clone spack，解压，把bin添加到环境变量和lib就可以使用spack了。

然后进hpctoolkit官网，挨着看怎么用spack安装它的。

因为之前折腾espreso，故gmake，gcc等等都已经升级达标了。

还是遇到一些问题：

1安装慢（只有靠等）

2报错说缺少patch。 yum install patch 解决

3报错说缺少fortran compiler。 spack compiler find。 得到compiler配置文件地址，vi进去，发现我的gcc9.3.1里面没有对应的fortran编译器。于是 yum search fortran，在得到的文件列表中发现其中一个develop-xxxx后边的介绍是支持gcc9.x的，yum install安装之。安装完自动配好了环境变量，解决。

4安装dyninst组件时卡住不动
